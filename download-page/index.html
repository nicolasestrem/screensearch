Frequently Asked Questions

Why does Windows SmartScreen show a warning?

ScreenSearch is currently signed with a self-signed certificate. Windows SmartScreen warns about apps from unknown publishers. This is normal for new open-source projects.

To proceed safely: Click "More info" then "Run anyway"

You can verify the download by checking SHA256 checksums or scanning with VirusTotal.

Is ScreenSearch safe to use?

Yes! ScreenSearch is open source and published under the MIT license. You can:

  • Review the source code on GitHub
  • Scan the installer with VirusTotal (0 detections)
  • Verify SHA256 checksums from the releases page
  • Build from source if you prefer

All data is stored locally on your machine. No telemetry or external connections except for optional model downloads.

What's the difference between Full and Lightweight installers?

Full Installer (460 MB): Includes the ONNX embedding model. Everything works offline immediately.

Lightweight Installer (11 MB): Downloads the model (449 MB) from HuggingFace on first run if you enable embeddings. Requires internet connection for initial model download.

Both installers provide the same functionality once set up.

How do I uninstall ScreenSearch?

Use Windows Settings > Apps > Installed Apps, find ScreenSearch, and click Uninstall.

The uninstaller will:

  • Remove the application files
  • Clean up AppData directory (%LOCALAPPDATA%\screensearch)
  • Remove Start Menu shortcuts
  • Ask if you want to keep your captured screenshots
Where are my screenshots stored?

By default, ScreenSearch stores data in:

  • Database: In the application directory (screensearch.db)
  • Screenshots: In the captures/ subdirectory
  • ONNX Model: In %LOCALAPPDATA%\screensearch\models\ (if downloaded)

You can configure these paths in config.toml

Can I use ScreenSearch on macOS or Linux?

Currently, ScreenSearch is Windows-only due to heavy use of Windows-specific APIs (WinRT, Win32, Windows OCR).

Cross-platform support would require significant architectural changes and is not planned for the near future.

How do I update ScreenSearch?

ScreenSearch checks for updates on startup and will notify you in the logs when a new version is available.

To update:

  1. Download the latest installer from this page or GitHub Releases
  2. Run the installer (it will detect and upgrade your existing installation)
  3. Your database and screenshots will be preserved

Troubleshooting

Installer won't run

Solution: Right-click the installer, select Properties, check "Unblock" at the bottom, click OK, then run again.

OCR not working

Solution: Install Windows OCR Language Pack from Settings > Time & Language > Language > Add a language > Install language features.

Web interface won't open

Solution: Check if port 3131 is available. You can change the port in config.toml. Ensure no firewall is blocking localhost connections.

High CPU or memory usage

Solution: Adjust capture interval in config.toml (default: 3000ms). Reduce frame_diff threshold or disable embeddings if not needed.
